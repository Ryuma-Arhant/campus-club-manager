# Campus Club Manager

A full-stack web application for managing campus clubs, events, and memberships.

## Tech Stack
- **Frontend:** React 18 + Vite + TailwindCSS + Framer Motion
- **Backend:** Node.js + Express + MySQL
- **Auth:** JWT with bcrypt password hashing

## Features
- **Students:** Browse clubs, request membership, RSVP to events
- **Club Admins:** Manage members, create events, assign roles
- **Super Admins:** Approve clubs, assign admins, view system stats

## Getting Started

### Prerequisites
- Node.js 18+
- MySQL 8+

### Backend
```bash
cd Backend
npm install
npm run dev
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
